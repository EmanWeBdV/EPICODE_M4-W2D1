## 🚀 Features

- **Photo search functionality**
  - Search images by keyword
  - Dynamic request to the Pexels API
  - Results rendered directly inside the page

- **Default content loading**
  - The app automatically loads images for the keyword **code** on startup

- **Dynamic DOM rendering**
  - Images are created with JavaScript
  - Results are inserted into the main `.row` container
  - Each result is displayed inside a responsive image box

- **Simple UI structure**
  - Search input field
  - Search action
  - Image grid layout

- **Educational Context**
  - Built as a frontend exercise to practice `fetch`, async functions, API integration and DOM manipulation
